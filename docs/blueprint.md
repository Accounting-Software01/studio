# **App Name**: Invoice Assistant

## Core Features:

- Invoice Fetching: Fetch invoice details from the Hari Industries API using the invoice ID.
- Data Formatting: Format dates and currency for display in the invoice.
- Print Invoice: Enable printing of the invoice with optimized layout for paper.
- Amount to Words: Convert the total amount to words for clarity on the invoice.

## Style Guidelines:

- Primary color: Deep Blue (#1A237E) to evoke trust and professionalism in financial transactions.
- Background color: Very light blue (#F0F4FF), close to white, provides a clean, professional background that will reduce printer ink consumption.
- Accent color: Warm Yellow (#FFB300) is analogous to the primary color, and highlights key details.
- Body and headline font: 'Inter', a sans-serif font, offers a modern and neutral look suitable for both headings and body text, promoting readability.
- Simple, professional icons from Lucide to indicate actions like printing and navigation.
- Clean, print-friendly layout with clear sections for company details, customer information, invoice items, and totals.
- Subtle loading animation while fetching invoice data to enhance user experience.
